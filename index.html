Exemplo de Parágrafo (tag <"p">). Serve para agrupar blocos de texto de forma organizada.
A Tag (<"br">) acima forçou esta frase a ir para a linha de baixo, mesmo estando dentro do mesmo parágrafo.
Podemos aplicar diferentes tipo de ênfase no texto: O uso da tag <<"strong">>" indica que este texto é MUITO IMPORTANTE
já a Tag <"em"> é usada para dar ênfase, deixando o texto em itálico. É ideal para termos técnicos ou estrangeiros.
